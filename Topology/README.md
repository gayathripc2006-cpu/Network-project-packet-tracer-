🌐 Network Topologies using Cisco Packet Tracer
📌 Overview

This project demonstrates basic network topologies using Cisco Packet Tracer to understand structure, communication, and performance.

🧠 Topologies
⭐ Star

All devices connect to a central switch.
+ Easy to manage
− Central device failure affects all

🔄 Ring

Devices form a circular connection.
+ No collision
− One failure breaks network

🚌 Bus

Single backbone cable connects all devices.
+ Low cost
− Backbone failure stops network

🕸️ Mesh

Every device connects to every other device.
+ Highly reliable
− Expensive and complex

🌳 Tree

Hierarchical (combination of star & bus).
+ Scalable
− Backbone dependency

🛠️ Tools
Cisco Packet Tracer
Routers, switches, PCs
🚀 How to Use
Open .pkt file in Cisco Packet Tracer
Use Simulation Mode
Test using ping
🎯 Outcome
Understand topology structures
Compare advantages & limitations
Practice network simulation
